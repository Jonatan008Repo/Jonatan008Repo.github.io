# Mi Portafolio Personal

Bienvenido a mi portafolio profesional. Este repositorio contiene mi sitio web personal donde muestro mis proyectos, habilidades y experiencia como desarrollador.

## 📋 Descripción

Portafolio profesional desarrollado con tecnologías modernas para mostrar mis proyectos, experiencia y habilidades técnicas. El sitio es completamente responsivo y optimizado para ofrecer una excelente experiencia de usuario.

## ✨ Características

- Diseño responsivo y moderno
- Animaciones fluidas y efectos visuales
- Sección de proyectos destacados
- Información profesional y contacto
- Rendimiento optimizado
- Interfaz intuitiva y accesible

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos avanzados y animaciones
- **JavaScript** - Interactividad y funcionalidad
- **tsParticles** - Efectos de partículas animadas
- **Git** - Control de versiones

## 🚀 Inicio Rápido

### Requisitos

- Navegador web moderno
- Git (para clonar el repositorio)

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Jonatan008/Jonatan008Repo.github.io.git

# Navegar al directorio
cd Jonatan008Repo.github.io

# Abrir en el navegador
# Simplemente abre el archivo index.html en tu navegador
```
