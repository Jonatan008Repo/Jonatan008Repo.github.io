¿QUE ES EL ERROR 404?

El error 404 es un código de estado HTTP enviado desde el servidor web al navegador, es decir, al cliente que hizo la petición HTTP. Es así como el navegador muestra el código como una página de error. El enlace que conduce a esta página se denomina "dead link" (enlace muerto) o "broken link" (enlace defectuoso).
